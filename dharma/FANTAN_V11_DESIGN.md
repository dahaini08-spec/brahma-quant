问题：
  当前系统单一出场策略 → 均盈3.23% 均亏2.97% → RR=1.09 → PF=0.59
  需要均盈提升到6%+，均亏压到2%以下

解决方案：根据信号强度选择出场轨道

┌─────────────────────────────────────────────────────────────────────────────┐
│  轨道A：极短线快出（1~4根K线）                                                │
│  适用：所有信号（PF=1.07，唯一正期望）                                        │
│                                                                             │
│  入场：限价挂单                                                               │
│  止损：入场K线最近摆动点（ATR×0.8）                                            │
│  止盈：入场后1H内 收益>0.8% 立刻出场                                           │
│      或 4根K线后 无论盈亏 出场                                                 │
│  目标：胜率55%+ / PF1.0+                                                    │
│  适合币种：SOL/SUI/BNB/DOGE（最优1~4H持仓标的）                               │
└─────────────────────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────────────────────┐
│  轨道B：当前体制长持（48H）                                                   │
│  适用：EXTREME信号 + 当前体制CHOP_HIGH                                        │
│                                                                             │
│  入场：限价挂单（RSI极端+4H中性+低ATR）                                         │
│  止损：入场K线结构低/高点（严格）                                                │
│  止盈T1：BB中轨（约2~3%）                                                     │
│  止盈T2：持有至48H（历史均收益+0.672%）                                        │

│  目标：让利润充分释放，均盈6%+                                                  │
│  适合币种：ATOM/TIA/LTC/DOT（最优24~48H持仓标的）                              │
└─────────────────────────────────────────────────────────────────────────────┘

双轨分配逻辑：
  信号产生时 → 评分≥80 + EXTREME + 体制=CHOP_HIGH → 轨道B（长持）
  信号产生时 → 评分50-80 或 快速标的 → 轨道A（快出）
  信号产生时 → 评分<50 → 丢弃

  做多：FR < -0.08% + RSI_1H < 45（市场付钱让你做多）
  做空：FR > +0.08% + RSI_1H > 55（市场付钱让你做空）
  优势：FR收益本身就是正收益，信号纯粹
  预期胜率提升：+5~8%（逻辑层面，待达摩院EXP-09验证）

通道C：OI异动通道（EXP-10待验证）
  触发：OI 4H变化率 > +15% + 方向确认
  做多：OI快速增加 + RSI<45（主动建多仓）
  做空：OI快速增加 + RSI>55（主动建空仓）
  优势：捕捉大资金入场信号

三通道优先级：
  通道B（FR套利）> 通道A（技术极端）> 通道C（OI异动）
  多通道共振时：信号质量×1.5，仓位×1.3

  陆 · 仓位系统 v3.0：动态Kelly
═══════════════════════════════════════════════════════════════════════════════

当前问题：固定仓位表（3%/4%/6%），不响应系统学习

新方案：动态Kelly仓位

基础Kelly计算：
  f = (W × RR - (1-W)) / RR × 0.25  # Quarter Kelly

参数来源（达摩院实时更新）：
  W = 最近100笔实盘胜率（初始用EXP-02历史值35.2%）
  RR = 当前双轨出场目标RR（轨道A=1.2，轨道B=3.0）

体制系数（来自EXP-02）：
  RECOVERY体制    → Kelly × 1.5（历史最优38%）
  CHOP_HIGH体制   → Kelly × 1.1（当前36%）
  BULL_ETF体制    → Kelly × 0.8（历史33%）
  BEAR_CRASH体制  → Kelly × 0   （激活黑天鹅模式）

信号质量系数（来自EXP-03）：
  CI下界>38%   → × 1.3
  CI下界36-38% → × 1.0
  CI下界34-36% → × 0.7
  CI下界<34%   → × 0（不开仓）

仓位上限：任何情况 ≤ NAV×6%
