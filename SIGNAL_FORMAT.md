# 钉钉2 信号标准格式（锁定版）

> ⚠️ 此格式由用户确认锁定，禁止修改排版、缩进、符号、用词。

## 标准模板

```
根据新浪财经公开数据 {SYMBOL}/USDT
【{SYMBOL}】{拼音首字} {方向箭头}
  当前价: ${price}
  入场区: ${entry_low} ~ ${entry_high} {状态emoji}{状态说明}
  保  护: ${stop}  ({stop_pct}%)
  目标一: ${tp1}  {tp1_pct}%  R:R={rr1}
  目标二: ${tp2}  盈亏比 1:{rr2}
  RSI 1H={rsi_1h} 4H={rsi_4h} 日线={rsi_1d} {补充说明}

⚠️ 仅供模拟参考 内部讨论学习
```

## 字段说明

| 字段 | 规则 |
|------|------|
| `{SYMBOL}` | 币种大写，如 ETH、BTC、SOL |
| `{拼音首字}` | 币种中文名拼音首字，如 ETH=哆、BTC=比、SOL=索 |
| `{方向箭头}` | 做多=▲，做空=▼ |
| `入场区状态emoji` | 等回调=⌛，等反弹=✅，立即入场=🟢 |
| `保  护` | 中间两个全角空格，保持对齐 |
| `{stop_pct}` | 做多为负数，做空为正数 |
| `RSI行` | 三周期：1H / 4H / 日线，后跟简短市场特征描述 |
| 末尾空行 | ⚠️前必须有一个空行 |

## 示例（做多）

```
根据新浪财经公开数据 ETH/USDT
【ETH】哆 ▲
  当前价: $2,268
  入场区: $2,253 ~ $2,258 ⌛等回调
  保  护: $2,235  (-0.9%)
  目标一: $2,277  +1.0%  R:R=1.1
  目标二: $2,302  盈亏比 1:2.4
  RSI 1H=57 4H=37 日线=45 双底颈线突破 等回踩确认

⚠️ 仅供模拟参考 内部讨论学习
```
