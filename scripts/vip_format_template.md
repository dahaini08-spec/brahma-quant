# 设计院 · 2026-06-20 封印
# 七条固定规则，任何AI输出均不得违反

## 固定模板格式

```
🌿 姓赵不宣  |  [标的] 今日布局

🔴  空单
等 [入场区] 反弹入场
止损 [SL]   目标 [TP1] / [TP2] / [TP3]
杠杆[X]x   仓[X]%

🟢  多单（轻仓）
等 [猎杀位] 被扫后 [入场区] 接
止损 [SL]   目标 [TP1] / [TP2]
杠杆[X]x   仓[X]%

⚠️ 主方向做空，多单博反弹严守止损
```

## 七条固定规则（永久不可违反）

1. 开头永远 🌿 姓赵不宣
2. 不出现"梵天量化"等系统名称
3. 空单在前主力，多单副方向轻仓
4. TP至少三档（空单TP1/TP2/TP3）
5. 多单网格接筹：先写猎杀位，再写接筹区
6. 杠杆与仓位必须同行标注
7. 末行固定：⚠️ 主方向做空，多单博反弹严守止损

## 数据来源规则

- 所有价格数据必须来自梵天 brahma_core 实时输出
- 禁止AI估算或捏造任何价格数字
- signal_id 供内部追溯，不出现在VIP格式正文
