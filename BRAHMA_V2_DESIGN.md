## 核心发现（五轮回测结论）

### 🔑 最重要：体制×方向矩阵

```
              BULL_TREND   RECOVERY    RANGING    BULL_PEAK   BEAR_TREND
  做多          ✅高胜率    ✅高胜率    ⚠️中等     ❌低胜率    ❌禁止
  做空          ❌绝对禁    ✅80.4%    ✅41.9%    ✅中等      ✅高胜率
```

**RECOVERY体制做空 WR=80.4%（n=46，跨标的验证）**
- 这是实盘 SHORT WR=83% 的真正来源
- 本质：BTC从底部反弹但均线未转 → 代币上涨是反弹顶 → 做空确定性极高
- 与具体标的无关，是市场结构性规律

### 🔑 ATR自适应止损的必要性

```
固定止损 5%:
  低波动币(BTC) → 止损过宽，亏损时太大
  高波动小币    → 止损过窄，频繁被扫
  MaxDD = 36.8%

ATR × 3.0 自适应:
  BTC: 止损 ≈ 3-4%（BTC波动小，止损自然紧）
  小币: 止损 ≈ 6-9%（小币波动大，止损自然宽）
  MaxDD = 9.1%  ← 降低 75%
```

## BRAHMA v2.1 信号逻辑（终极版）

### 做多条件
```python
LONG = (
    RSI_1H < 25              # 灵魂库极超卖阈值
    RSI_4H < 50              # 4H未超买
    MACD 金叉                 # 动能转向
    BB_POS < 0.25            # BB低位（低于均值25%区间）
    ATR_PCT > 0.33%          # 灵魂库最低ATR阈值（有效波动）
    体制 in [BULL_TREND, RECOVERY]  # 大方向对
    RSI_1D < 55              # 日线不过热
)
```

### 做空条件（双轨道）
```python
# 核心武器：RECOVERY体制反弹顶（WR=80%+）
SHORT_RECOVERY = (
    RSI_1H > 65              # 相对超买（门槛比RANGING低）
    RSI_4H > 52
    MACD 死叉
    BB_POS > 0.70
    体制 == RECOVERY          # 仅限此体制
)

# 补充：RANGING/BULL_PEAK震荡顶（WR≈42-50%）
SHORT_RANGING = (
    RSI_1H > 75              # 灵魂库极超买阈值
    RSI_4H > 56
    MACD 死叉
    BB_POS > 0.76
    BB_WIDTH > 0.06          # 灵魂库bb_width_min
    ATR_PCT > 0.33%
    RSI_1D > 50              # 日线不超卖（防误做空反弹）
    体制 in [RANGING, BULL_PEAK]
)
```
